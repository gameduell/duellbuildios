## Description

Use this plugin to build for the ios platform.
## Usage:
`$ duell build ios -simulator -debug`
## Arguments:
* `-simulator` &ndash; runs the app on the ios simulator.
* `-debug` &ndash; Use this argument if you want to build in debug.
